# Roadmap

## Short term
- Improve oven status mapping for dual-cavity ranges.
- Add device health and connectivity indicators.
- Add preheat presets (Bake 350, Air Fry 400, etc.).

## Mid term
- Add ThinQ device categories (washer, dryer, refrigerator).
- Add cooktop controls with safety gating.
- Add per-device permissions and lockouts.

## Long term
- Multi-vendor integrations (e.g., LG TV, Bose soundbar).
- Mobile-friendly dashboard and kiosk mode.
- User auth + multi-home support.
